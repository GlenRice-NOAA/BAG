Building with Linux gcc

Configure and build bag

To generate the Makefiles, simply go to the directory you untarred the tarball into, and type:

cmake -G "Unix Makefiles" -B build -S . -DCMAKE_BUILD_CONFIG=Release -DCMAKE_INSTALL_PREFIX=/usr/local

To add support for Python, SWIG 4.0.1 or newer is needed. Add -DBAG_BUILD_PYTHON=ON to the above command.

The makefile is now available in the bag/build folder. To build, simply type:

cmake --build build

To install, simply type:

sudo cmake --install build
